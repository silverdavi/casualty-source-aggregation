# Data

81 conflicts, 1900--2026, with per-side military / civilian-direct / indirect
death counts as low--high ranges with cited sources.

## Files

| File | Purpose |
|---|---|
| `wars_input.json` | Curated list of 81 conflicts (id / name / dates / region). The input that was researched. |
| `per_war/<war_id>.json` | One file per conflict, schema below. |
| `summary.json` | Machine-readable index aggregating all 81 conflicts. |
| `totals.csv` | Flat CSV: `war_id, war_name, side, bucket, low, high, notes`. |
| `SUMMARY.md` | Human-readable rolled-up tables (regenerated by `analysis/summarize.py`). |

## Per-war JSON schema

Each `per_war/<war_id>.json` is a single object with this structure (every
numeric field can be `null` if not securely known):

```jsonc
{
  "war_id":     "<snake_case identifier>",
  "name":       "<canonical name>",
  "start_year": 1939,
  "end_year":   1945,            // null if ongoing
  "ongoing":    false,
  "regions":    ["Europe", "North Africa", "Atlantic"],
  "summary":   "<2-4 sentence summary>",

  "sides": [
    {
      "name":    "Allies",
      "members": ["United States", "United Kingdom", ...],

      "military_killed": {
        "low": ..., "high": ..., "point": ...,
        "notes":   "<string>",
        "sources": ["<url>", ...]
      },

      "civilians_killed_directly": {
        "low": ..., "high": ..., "point": ...,
        "breakdown": [
          {
            "cause":         "bombing | shelling | massacre | execution | ...",
            "victims_group": "<who, e.g. 'Japanese civilians (firebombing)'>",
            "low": ..., "high": ...,
            "notes":   "<string>",
            "sources": ["<url>", ...]
          }
        ],
        "notes":   "<string>",
        "sources": ["<url>", ...]
      },

      "deaths_from_actions": [
        {
          "type":          "camp | famine | blockade | disease | displacement | "
                           "forced_labor | ethnic_cleansing | scorched_earth | ...",
          "victims_group": "<who>",
          "low": ..., "high": ..., "point": ...,
          "notes":   "<string>",
          "sources": ["<url>", ...]
        }
      ]
    }
    // ... one object per belligerent side
  ],

  "totals": {
    "military_low": ..., "military_high": ...,
    "civilian_low": ..., "civilian_high": ...,
    "grand_low":    ..., "grand_high":    ...,
    "notes":        "<how the rolls-up handle indirect deaths and double-counting>"
  },

  "key_atrocities": [
    {
      "name":          "<event name>",
      "year":          1937,
      "perpetrator":   "<side>",
      "victims_group": "<who>",
      "low": ..., "high": ...,
      "notes":   "<string>",
      "sources": ["<url>", ...]
    }
  ],

  "uncertainty_notes": "<paragraph on what makes this conflict's totals contested>",

  "sources": [
    {"url": "<url>", "title": "<title>", "publisher": "<organisation>"}
  ]
}
```

## The three death buckets per side

* **A. `military_killed`.** That side's own combatants killed by *any* cause
  (combat, disease, accidents, POW deaths, executions, friendly fire). Not
  enemy combatants killed by that side.
* **B. `civilians_killed_directly`.** Civilians killed *directly* by that
  side's forces (bombing, shelling, massacre, summary executions, deliberate
  terror campaigns). Sub-broken-down by `cause` × `victims_group`.
* **C. `deaths_from_actions`.** Deaths caused *indirectly* by that side's
  policies and actions: concentration / extermination camps, deliberate
  famine, blockade starvation, siege disease, forced labour, deportation,
  ethnic cleansing, scorched earth. This bucket is the entire reason the
  dataset is structured per-side: most aggregate casualty tables collapse
  it into "civilians", erasing the policy-attribution signal.

For events where the literature does not allow clean attribution between
sides or buckets, the `notes` field says so explicitly rather than guessing.

## Coverage

The `wars_input.json` list aims for major lethality plus historical
importance, with explicit inclusion of:

* The two world wars and their major theatres (incl. the Holocaust as a
  separate entry from WWII Europe so the indirect-deaths bucket is not
  shadowed).
* 20th-century civil wars and post-colonial conflicts (Russian Civil War,
  Spanish Civil War, Chinese Civil War, partition of India, Indochina,
  Algerian War, Biafra, Bangladesh, Khmer Rouge, Iran--Iraq, ...).
* Late-20th-century genocides and mass-killing events (Bangladesh,
  Indonesia 1965--66, East Timor, Cambodian genocide, Rwanda, Bosnia,
  Kosovo, Darfur, Rohingya, Tigray).
* Recent and ongoing conflicts (Russia--Ukraine 2022; Gaza / Israel 2023;
  Sudan 2023; Yemeni civil war; Syrian civil war; Mexican drug war;
  Myanmar; Somalia; Boko Haram; the Iraq War 2003 and the anti-ISIS war).

The list is not exhaustive. Smaller wars and short border clashes are
omitted by design; the dataset is built around conflicts where the
under-fudgible relations of the paper can plausibly bind.
