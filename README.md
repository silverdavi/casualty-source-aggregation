# Casualty Source Aggregation

Companion repository for the manuscript

> *Bounds and Estimators for the Civilian-to-Combatant Death Ratio
> from Aggregation of Conflicting Sources.*

When casualty figures are reported by sources with opposing political incentives,
naive aggregation is biased in a direction that depends on which side the
analyst trusts more. This repository provides:

1. A **structured dataset** of per-side death tolls (military / civilian-direct /
   indirect) with low-high ranges and inline source URLs for **81 major
   conflicts, 1900--2026**.
2. A **spatial Bayesian simulator** for the Gaza war 2023--2026 that produces a
   joint posterior over the militant share, civilian adult-male deaths, and
   women-and-children deaths from independently measured demographic and
   manpower facts.
3. The **paper LaTeX source** (master manuscript, modular section files,
   bibliography, publication figures, and supplement).

## Per-war JSON schema

Each file in `data/per_war/` is a single dictionary. The full schema is
documented in `data/README.md`; the high-level fields are:

```json
{
  "war_id":       "<snake_case identifier>",
  "name":         "<canonical name>",
  "start_year":   1939,
  "end_year":     1945,
  "ongoing":      false,
  "regions":      [...],
  "summary":      "<2-4 sentence summary>",
  "sides": [
    {
      "name":    "<belligerent>",
      "members": [...],
      "military_killed":          {"low":..., "high":..., "point":..., "notes":..., "sources":[...]},
      "civilians_killed_directly":{"low":..., "high":..., "point":..., "breakdown":[...], "notes":..., "sources":[...]},
      "deaths_from_actions": [
        {"type": "camp|famine|blockade|disease|displacement|forced_labor|...",
         "victims_group":..., "low":..., "high":..., "notes":..., "sources":[...]}
      ]
    }
  ],
  "totals": {"military_low":..., "military_high":..., "civilian_low":...,
             "civilian_high":..., "grand_low":..., "grand_high":...,
             "notes":...},
  "key_atrocities":      [...],
  "uncertainty_notes":   "...",
  "sources":             [{"url":..., "title":..., "publisher":...}]
}
```

The three death buckets per side are:

* **A. `military_killed`** -- that side's own combatants killed (any cause:
  combat, disease, executions, POW deaths).
* **B. `civilians_killed_directly`** -- civilians killed *directly* by that
  side's forces (bombing, shelling, massacre, summary executions, terror
  campaigns), with sub-breakdown by cause and victim group.
* **C. `deaths_from_actions`** -- deaths caused *indirectly* by that side's
  policies (concentration / extermination camps, deliberate famine, blockade
  starvation, siege disease, forced labour, ethnic cleansing, scorched
  earth). The Holocaust, Holodomor, Bengal 1943, Biafra, Khmer Rouge,
  Yemen blockade, Tigray, and Gaza all illustrate why this bucket cannot be
  collapsed into the civilian-direct bucket.

Where the literature cannot cleanly attribute deaths between sides or
buckets, the `notes` field flags this; otherwise estimates are given as
low--high ranges with cited sources rather than single point estimates.

## Quickstart

```bash
# 1. Set up
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. Re-aggregate from data/per_war/ into summary.json / SUMMARY.md / totals.csv
python analysis/summarize.py

# 3. Build the headline 5x3 figure
python analysis/figure.py --out master_figure
python analysis/figure.py --out master_figure_recent50 --years-since 50 --min-deaths 1000

# 4. First-pass demographic Bayes for Gaza (writes figures/gaza_bayesian.{pdf,png})
python analysis/gaza_bayesian.py

# 5. Spatial Bayesian Gaza simulator (writes posterior.npz + posterior_figure)
python gaza_sim/01_simulator.py
python gaza_sim/02_figure.py
python gaza_sim/03_report_and_apply.py

# 6. Build the paper from source
cd paper
pdflatex paper && bibtex paper && pdflatex paper && pdflatex paper
# or, regenerate publication figures + supplement first:
python build_figures.py
python build_supplement.py
```

## Reproducing the paper's headline numbers

Every bound, contradiction radius, and table in the paper is generated and
cross-checked by

```bash
python3 analysis/validate_bounds.py   # exits non-zero if any check fails
```

which writes `analysis/validation_report.json` plus the two LaTeX tables
(`paper/content/tab_sensitivity.tex`, `paper/content/tab_convergence.tex`).
The mathematical claims are verified at three levels:

```bash
python3 analysis/verify_proofs.py         # sympy + Monte-Carlo checks of every theorem
cd formal/casualty_proofs && lake build   # Lean 4 / mathlib machine-checked proofs
```

The Lean development (`formal/casualty_proofs/CasualtyProofs.lean`, no
`sorry`s) kernel-checks the core algebra: the identification inversions
(eq. 2 and Theorem 3.3), monotonicity of $q(\mu)$, all three closed forms
of the $\eta$-slack bias and its sign (Remark 3), the manpower bound, the
monotonicity of feasibility in $M$ behind Theorem 4.3's one-sided penalty,
and the Section 6 arithmetic in exact rational form (including a formal
refutation of a misprinted table value). `docs/proof_verification.pdf` is
the human-readable dossier with full proofs and an error log of everything
caught by the verification pipeline and by external review.
Key outputs (Ministry of Health demographic anchor, males 18+ convention):

* Identified set for the Gaza combatant share: $q \in [0, 25.1\%]$ with no
  exposure assumption ($\mu \ge 1$), $q \in [0, 6.3\%]$ under the
  historically calibrated exposure range $\mu \in [2, 3.5]$.
* IDF claim of 17--25k combatants killed ($q \approx 24$--$36\%$ over 70k):
  the 25k endpoint is infeasible for any $\mu \ge 1$; contradiction radii
  range from $\rho \approx 8$ SE (agnostic exposure, MoH anchor) to
  $\rho \approx 31$ SE (calibrated exposure, OHCHR anchor). The rejection
  survives undercount-corrected death tolls (94k--106k) and a 5% Huber
  contamination allowance on every input.
* Spatial Bayesian posterior (one prior-dependent point *within* the set;
  fully documented in the paper's Appendix B): $q = 2.5\%$, 95% credible
  interval $[1.6\%, 3.8\%]$, implied civilian-to-combatant ratio
  $\sim 43{:}1$ $[30, 64]$ (gaza_sim/report.md, gaza_sim/posterior.npz).

## Caveats

* War-casualty figures are **fundamentally contested**. The same event has
  order-of-magnitude disagreements between serious scholars (Bengal 1943:
  0.8M--3.8M; Biafra: 0.5M--3M; the 1958--62 China famine: 15M--55M;
  Russian Civil War: 5M--12M). The `low`--`high` ranges in this dataset
  reflect the spread of credible scholarly estimates, not arithmetic
  precision.
* The direct/indirect split is sometimes unrecoverable -- many estimates
  lump everything together. The `notes` fields call this out where it
  matters.
* Summing the `total_high` column in `data/SUMMARY.md` (~278M)
  double-counts overlapping conflicts (WWII theaters vs the Holocaust, Iraq
  War vs the anti-ISIS war) and should not be read as a clean grand total
  for 20th--21st century war.
* For the still-active wars (Russia--Ukraine, Gaza/Israel, Sudan, Yemen,
  Mexico, Myanmar, Somalia, Boko Haram) the toll is still moving; see each
  per-war JSON for the cited as-of date.
